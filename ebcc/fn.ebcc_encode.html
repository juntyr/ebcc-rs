Expand description

Encode climate variable data using EBCC compression.

§Arguments

  • data - Input data as a slice of f32 values
  • config - EBCC configuration parameters

§Returns

A vector containing the compressed data bytes.

§Errors

Returns an error if:

  • Configuration is invalid
  • Input data size doesn’t match configuration dimensions
  • Compression fails
  • Memory allocation fails

§Examples

use ebcc::{ebcc_encode, EBCCConfig, ResidualType};
use ndarray::Array;

// 2D ERA5-like data: 721x1440
let data = Array::from_shape_vec((1, 721, 1440), vec![1.0f32; 721 * 1440]).unwrap();
let config = EBCCConfig::max_absolute_error_bounded(30.0, 0.01);

let compressed = ebcc_encode(data.view(), &config)?;
println!("Compressed {} bytes to {} bytes",
         data.len() * 4, compressed.len());
